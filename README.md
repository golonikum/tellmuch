# tellmuch

## About

Web application for helping in translating texts and remembering new words. 

## Technologies

1. Back-end:
  * Node.js (express)
  * WebSockets
  * Mongoose
2. Front-end:
  * React & RxJS
  * Bootstrap
  * SASS
3. Package-manager:
  * WebPack
4. Developing:
  * Heroku
5. Authentication:
  * VK API [https://vk.com/dev/authcode_flow_user]
